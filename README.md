# Hi 👋 My name is Jeremy

## Python Developer who loves automation

**IT Operations Engineer & Automation Architect** *14 years of building, breaking, and automating systems*

I develop Python solutions to automate processes and leverage artificial intelligence by integrating both local open-source large language models (LLMs) and cloud-based language models into effective tools. My work encompasses everything from concise utilities that improve daily workflows to sophisticated, multi-step pipelines for task automation, data processing, AI-driven operations, and management of self-hosted infrastructure.

> **Fundamental principle:** Any process that is manual and repeatable should be automated. Once automated, processes ought to be maintainable. When maintainable, they should be designed for scalability.
